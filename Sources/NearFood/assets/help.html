sFood v1.0

sFood is an application designed to help users search for fast food restaurants that near their location.

The main functions:

  • The first screen the user has two options: one is logging in account to use the advanced features of the system, second is accessing right into the application without logging in will limit the features.
  • The first function is Search to find the location.
  • The second function is Check-in to help users check-in one location on the server.
  • The third function is Favorite to save your locations .
  • The fourth function is Setting consists of information on the application, guidelines, registration, log in where the user is not logged in yet or do not have an account and profile, or changing password when login.
